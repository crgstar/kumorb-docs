Kumorb

Política de Privacidad

Introducción

Kumorb es una aplicación de pronóstico del tiempo especializada en visualización gráfica. Este documento explica cómo manejamos la información en esta aplicación.

Importante: Esta aplicación no almacena información personal en servidores ni dispositivos.

Información que Recopilamos

Datos de Ubicación

  • Momento de Recopilación: Solo mientras se usa la aplicación, obtenida cada vez
  • Almacenamiento: No se almacena en servidores ni dispositivos
  • Propósito: Únicamente para mostrar información meteorológica de su ubicación actual

Los datos de ubicación se descartan inmediatamente después de obtener los datos meteorológicos.

Servicios de Terceros

Open-Meteo API

Utilizamos la API de Open-Meteo para obtener datos meteorológicos. La información de ubicación (latitud y longitud) se envía a este servicio.

Términos de Servicio de Open-Meteo: https://open-meteo.com/en/terms

Publicidad y Compras dentro de la Aplicación

  • Publicidad: No se muestra ninguna
  • Compras dentro de la Aplicación: Actualmente ninguna (puede cambiar en el futuro)

Audiencia Objetivo

  • Clasificación por Edad: Todas las edades

Cambios en Esta Política

Esta política puede actualizarse según sea necesario. Los cambios significativos se notificarán a través de la aplicación o en esta página.
