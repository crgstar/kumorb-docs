Kumorb

プライバシーポリシー

はじめに

Kumorb（クモーブ）は、グラフを軸にした天気予報アプリです。 このプライバシーポリシーでは、アプリがどのように情報を扱うかを説明します。

重要：Kumorbは、個人情報をサーバーや端末に保存しません。

収集する情報

位置情報

  • 収集するタイミング：アプリを使用中のみ、都度取得します
  • 保存について：サーバーにも端末にも保存しません
  • 利用目的：現在地の天気情報を表示するためのみ使用します

位置情報は天気データ取得のためにのみ使用され、取得後すぐに破棄されます。

外部サービスとの連携

Open-Meteo API

天気データの取得のために、Open-Meteo API を使用しています。 このサービスには、お客様の位置情報（緯度・経度）を送信します。

Open-Meteo のプライバシーポリシーについては、以下をご確認ください：
https://open-meteo.com/en/terms

広告・課金について

  • 広告：Kumorbには広告は一切表示されません
  • 課金：現在、アプリ内課金はありません（将来的に検討する可能性があります）

対象年齢・地域

  • 対象年齢：全年齢対象
  • 対象地域：現在は主に日本国内向けにサービスを提供しています

プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて変更される場合があります。 重要な変更がある場合は、アプリ内またはこのページでお知らせします。
